# Rasberry Pi Pico Media Controller

## this alows you to controll music volume and stop/play/forward music

## Pinout:
- GP16 -> play / stop button
- GP17 -> shuffle button
- GP18 -> forward
- GP19 -> backward
- GP20, GP21 -> rotary encoder

tie other ends to GND
